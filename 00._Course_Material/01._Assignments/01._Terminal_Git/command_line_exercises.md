## How to solve the exercises

You will find the instructions in the README after you have accepted the assignment and it has been forked to your user.

## How to hand in

1. Accept the assignment in Github Classroom (link above)

2. Clone the repository to your local machine

3. Solve the exercises in the files given

4. Commit and push your changes to your forked repository

5. This will automatically submit your assignment
