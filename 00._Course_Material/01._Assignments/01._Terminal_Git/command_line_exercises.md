# Command Line Exercises

Solve the basic Javascript exercises given in the repository.

**Hand-in**: Submit via GitHub Classroom: https://classroom.github.com/a/0EijYl1S

Try the above method first. There is a bug in GitHub that means that some of you can't hand in. In that case, hand it in the Teams room under Assignments.

**Deadline**: Check the deadline in GitHub Classroom. Especially the time. The deadline is the cutoff.

---

## How to solve the exercises

You will find the instructions in the README after you have accepted the assignment and it has been forked to your user.

---

## How to hand in

1. Accept the assignment in Github Classroom (link above)

2. Clone the repository to your local machine

3. Solve the exercises in the files given

4. Commit and push your changes to your forked repository

5. This will automatically submit your assignment
