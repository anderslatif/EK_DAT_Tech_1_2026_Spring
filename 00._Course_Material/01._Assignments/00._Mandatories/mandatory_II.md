# The task

In addition to fulfilling the Azure deployment goal for Ønskeskyen, make sure that you fulfill the learning goals.

Write down all the learning goals from the column titled `learning goals` in the semester plan.

For every point that contains the word "can", make sure that you can do that with a snippet or example. Then you can write: "I can do it!".

For every point that is based around "knows" / "understands" or similar, write a couple sentences or keywords down that could help you explain it.

Basically, make sure that you fulfill the learning goal for the exam.
